Virtual Sound

Demo
GitHub

Written in JavaScript, Virtual Sounded is built in Reactjs; built by Gulp; served on Node / Express; and its data is managed by MongoDB.
Initially created for "fun-loving dads" who want to experiment with unusual sound bites, the app binds any short sound file to any keyboard key. It may be used to add pizzazz to your presentation or make your friends smile.
