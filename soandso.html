Tech Stack

So&So is an MVP inspired by tlk.io. It is a super clean, no-frills chat application for users who need a neutral space to meet and talk while maintaining anonymity.
JavaScript - AngularJS - Node - Express - Bcrypt - Knex - SQLite3
